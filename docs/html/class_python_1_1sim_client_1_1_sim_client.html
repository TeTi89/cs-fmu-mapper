This client connects to the OPCUA Server of the Modelica interactive Simulation. A interactive simulation step is triggered by setting 'step' true. The client writes input values to the simulation, 
then it triggers a simulation step and waits the configured 'sleeptime' before reading the output values. To ensure a proper time scaling the 'stepSize' of the Modelica Simualtion has to be adjusted to match
the actual period time of the read-write process. For matching purposes the mean and std of the actual execution time is calcuated iterativley when the parameter 'doPeriodStats' is set to true. The
calcuclated values are printed every 1000 execution cycles. Ensure that the 'stepSize' of the Modelica simulation is matched with the mean actual execution time and the Plc cylce set to mean actual execitution time
plus at least five times std of the actual exection time.

Args:
    config (dict):  'SimClient' section of the configuration
    lock (asyncio.Lock):  A shared Lock object to ensure consistent read and write operations.
